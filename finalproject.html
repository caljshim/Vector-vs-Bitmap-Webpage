Bitmap vs Vector Graphics

What are bitmap and vector images, and how are they different?

A bitmap (or raster) graphic is created from rows of colored pixels that together form an image. Raster files support millions of different colors and shades for use in any kind of image. The three most common file formats for rasters are JPEG, GIF, and PNG. Some of the less common ones are TIFF, XBM, BMP, and PCX. Rasters represent the majority of images found online and are much more common than vectors. The image displayed on computer monitors is also a raster, as are the outputs of printers, scanners, and similar devices.

A vector (or an "object oriented") graphic is constructed using mathematical formulas describing shapes, color, and placement. Unlike a bitmap, a vector contains instructions about where to place each of the image's components. Some vector file formats include SVG, EPS, and PDF. Vectors are most often used for creating logos, line art, 3D-like renderings, and animations.

Uses
Both image types are useful in their own right, making neither one or the other superior to the other. For example, raster images are much better suited for photographs, brochures, or advertisements because rasters are more detailed at a fixed resolution. In the three examples named before, re-sizing or transforming the resolution isn't usually necessary. Vectors, however, are best suited for things like logos, text, or line-art because they can be scaled to any size without losing quality. Logos, for example, are very likely to be changed to fit multiple sizes by a company.

Vector and Bitmap Summary

Bitmap (Raster)

  • Pixel-based
  • Raster programs best for editing photos and creating continuous tone images with soft color blends
  • Do not scale up optimally - Image must be created/scanned at the desired usage size or larger
  • Large dimensions & detailed images equal large file size
  • It is more difficult to print raster images using a limited amount of spot colors
  • Some processes cannot use raster formats
  • Depending on the complexity of the image, conversion to vector may be time consuming
  • Raster images are the most common image format, including: jpg, gif, png, tif, bmp, psd, eps and pdfs originating from raster programs
  • Common raster programs: photo editing / paint programs such as Photoshop & Paint Shop, GIMP (free)

Vectors

  • Shapes based on mathematical calculations
  • Vector programs best for creating logos, drawings and illustrations, technical drawings. For images that will be applied to physical products.
  • Can be scaled to any size without losing quality
  • Resolution-independent: Can be printed at any size/resolution
  • A large dimension vector graphic maintains a small file size
  • Number of colors can be easily increased or reduced to adjust printing budget
  • Vector art can be used for many processes and easily rasterized to be used for all processes
  • Can be easily converted to raster
  • It is not the best format for continuous tone images with blends of color or to edit photographs
  • Common vector graphic file format: ai, cdr, svg, and eps & pdfs originating from vector programs
  • Common vector programs: drawing programs such as Illustrator, CorelDraw, Inkscape (free)
